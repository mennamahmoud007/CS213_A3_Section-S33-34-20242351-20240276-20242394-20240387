Detailed Description

Special Tic-Tac-Toe variant where the board never fills.

In this mode, each player keeps placing symbols. After every 3 moves, the oldest placed symbol gets removed automatically. This creates an "infinite" gameplay loop where the board keeps changing.

The class tracks moves using a queue and checks win conditions based on the current board state.
